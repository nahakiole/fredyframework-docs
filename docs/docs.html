Documentation

The Fredyframework is a PHP framework intended to ease a developers life.

The project is hosted on Github and is available for use under the MIT software license. You can report bugs and discuss features on the GitHub issues page, or send tweets to @fredyframework.

Requirements

PHP 5.4
MySQL 5.6
Apache 2.4

Get started

PHP

        
			
namespace View;


use Controller\Navigation;
use Fredy\View\HTMLResponse;

class FrontendResponse extends HTMLResponse {

    function __construct($templatePath, $request)
    {
        parent::__construct($templatePath);
        $navigation = new Navigation($request);
        $this->setTwigVariables(['Navigation' => $navigation->getNavigation('navigation.yaml')]);
    }
}
